# ttf2png (a tool for parsing font to glyph image)

ttf2png python provides parsing single glyph from font file and saving image file.

## Usage example

```python
ttf_file_path = os.path.join( cwd,"ttf\\OCR-B-10 BT.ttf")
font_size = 48
```
